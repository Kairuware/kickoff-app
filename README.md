# ⚽ KickOff — World Cup 2026 Alarm App

Never miss a World Cup 2026 kickoff — automatic match alarms in your local timezone.

🌍 **[Download the App](https://kairuware.github.io/kickoff-app#download)**
📄 **[Privacy Policy](https://kairuware.github.io/kickoff-app/privacy.html)**

## Features
- ⏰ All 104 WC2026 fixtures with automatic timezone conversion
- 🔔 Custom alarms — 5, 10, 15, 30, 60 minutes before kickoff
- 🔍 Search and filter by team, group, stage
- 🌍 Works in every timezone worldwide
- ⚡ Lightweight, offline, no ads, no account needed
